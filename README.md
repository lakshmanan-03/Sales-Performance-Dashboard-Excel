# Sales Dashboard using Excel

## Overview

Designed an interactive sales dashboard using Microsoft Excel to analyze business performance across revenue, profit, category, payment mode, and geographic regions.

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* KPI Cards
* Data Cleaning
* Slicers

## Key Metrics

* Total Sales: ₹6.18M
* Total Profit: ₹1.61M
* Total Orders: 1194
* Total Quantity
* Profit Margin

## Dashboard Insights

* Monthly Sales Trend
* Monthly Profit Trend
* Sales by Category
* Profit by Category
* Sales by Payment Mode
* Top 5 States by Sales
